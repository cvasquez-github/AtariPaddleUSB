#include "Joystick2.h"

# AtariPaddleUSB
Using ATARI Paddle Controllers in a PC via USB

## Atari Paddle Controlles DB9 connector:
| Joystick        | DB9             | Comment          |  
| :-------------: | :-------------: |  :-------------: |  
| PAD 1 BUTTON    | PIN 3           | Joystick LEFT |
| PAD 2 BUTTON     | PIN 4          | Joystick RIGHT |
| GROUND GND      | GND PIN 8       |
| 5V+             | PIN 7           |
| PAD 1 INPUT     | PIN 9           |
| PAD 2 INPUT     | PIN 5           |
