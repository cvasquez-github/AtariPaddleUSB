# AtariPaddleUSB
Using ATARI Paddle Controllers in a PC via USB

| Joystick        | DB9             | Comment             |  
| :-------------: | :-------------: |              |  
| PAD 1 BUTTON    | PIN 3           | Regular Joystick LEFT |
| PAD 2 BUTTON     | PIN 4          | Regular Joystick RIGHT |
| GROUND GND      | GND PIN 8       |
| 5V+             | PIN 7           |
| PAD 1 INPUT     | PIN 9           |
| PAD 2 INPUT     | PIN 5           |
